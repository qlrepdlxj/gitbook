## gitbook 설치 및 사용법

#### node.js 설치

>sudo apt-get install nodejs

#### npm 설치

>sudo apt-get install npm

#### gitbook-cli 설치

>npm install -g gitbook-cli

#### gitbook 설정
> 파일이 있는 해당 디렉토리 내에서 작업하는것이 좋다.

>gitbook init 폴더이름

>gitbook serve
